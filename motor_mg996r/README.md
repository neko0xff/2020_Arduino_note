# motor_mg996r
## 功能
* 類型:伺服馬達(servo motor)
## 規格
* 重量 : 55g
* 角度：180度 (對應的角度是-90度~+90度)
* 工作電壓 : 4.8~7.2V
* 拉力 : 9.4kg/cm(工作電壓=4.8V ), 11kg/cm(工作電壓=6V)
* 速度 : 0.17sec/60degree(4.8v) , 0.14sec/60degree(6v)
## 接腳
* 橘 : 訊號(PWM)
* 紅 : VCC(5V)
* 褐 : GND
