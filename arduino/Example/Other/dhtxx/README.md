# dhtxx
## 功能
* 溫濕度模組
## 函式庫
* [DHT.h](https://github.com/adafruit/DHT-sensor-library)
## dht11 規格 
* 濕度測量範圍：20~90%
* 濕度測量精度：±5%
* 溫度測量範圍：0~50℃
* 溫度測量精度：±2℃
* 電源供應範圍： 3~5V
* 頻率不可超過：0.5Hz (每2秒一次)
## 接腳(由左開始)
* VCC
* DATA
* NC  => 空腳(不必接)
* GND
